Atrybut UIButton
Użycie atrybutu na viewmodelu pozwala na osadzenie przycisku na dialogu.
Atrybut należy dodać do właściwości typu string przypisując jako wartość nazwę metody, która ma się wywołać po kliknięciu na przycisk.
Przykład wykorzystania
Kod modelu.
Podczas inicjalizacji z przyciskiem zostaje powiązania metoda A kliknięcie w przycisk przełącza powiązaną metodę na B itd.
[UIController]
public class ButtonExample : TreeRelationModel
{
    private string _commandName = nameof(A);

    [UIButton]
    public string CommandName
    {
        get => _commandName;
        set => SetValue(nameof(CommandName), ref _commandName, value);
    }


    public void A(ButtonExample vm)
    {
        vm.CommandName = nameof(B);
    }
    public void B(ButtonExample vm)
    {
        vm.CommandName = nameof(A);
    }
}
