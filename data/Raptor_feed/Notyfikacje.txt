# 	Notyfikacje
## Opis

W systemie VENDO.ERP istnieje możliwość podpięcia i skonfigurowania funkcjonalności związanej z notyfikacjami. Notyfikacje mogą być realizowane za pomocą:
-	wysyłanych maili
-	wiadomości w VENDO.CHAT
-	wiadomości SMS 


Notyfikacje mogą być one kierowane do wybranych osób, które będą mogły podjąć kroki związane z zarządzaniem tymi obszarami. 
Ustawienie wysyłanych notyfikacji w postaci wiadomości mailowych czy i VENDO.CHAT nie wymaga dodatkowych rozwiązań zewnętrznych. W przypadku SMS-ów potrzebne jest wykupienie usługi jednego z operatorów bramki SMS.

System VENDO.ERP zapewnia szereg możliwości reakcji na występujące zdarzenia, które wymieniono w poniższej tabeli.

| Kontrola przy zamykaniu dokumentu    | -                                               |
| Kontrola przy zapisie pozycji dokumentu | -                                               |
| List Przewozowy                      | Eksport listu przewozowego                      |
| Logi                                 | -                                               |
| MES                                  | Po agregacji (SCADA), Po zmianie stanu, Przed agregacją (SCADA), Przed zapisem |
| Obiekty                              | Utworzenie obiektu, Zmiana wartości dowolnej obiektu |
| Plan zlecenia                        | Zmiana statusu, Zmiana terminu zakończenia      |

| Rejestracja operacji KKW             | Rozpoczęcie rejestracji KKW, Zakończenie rejestracji KKW |
