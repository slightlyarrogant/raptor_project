Kolory w Vendo
Klasa Amag.UI.VendoColors zawiera listę kolorów, która odpowiada tej z Bootstrap.
Podstawowe kolory:

Blue
Indigo
Purple
Pink
Red
Orange
Yellow
Green
Teal
Cyan
Gray
GrayDark
White
Black

Kolory "funkcyjne":

Primary
Secondary
Success - np. poprawnie zakończona operacja
Danger - np. błąd lub poważny problem
Warning - ostrzeżenie
Info - informacja
Light
Dark

Określenie koloru etykiety na dialogu
Na stałe (enum Amag.UI.VendoColorName zawiera listę kolorów "funkcyjnych"):
[UILabel(Color = VendoColorName.Danger)]
public string Property { get; set; }

Bindowany:
[UILabel(ColorProperty = nameof(PropertyLabelColor))]
public string Property { get; set; }
public VendoColor PropertyLabelColor => VendoColors.Purple;

Określenie koloru pola edycyjnego
Ukreślenie koloru przez suffix:
public string Property { get; set; }
public VendoColor PropertyColor => VendoColors.Warning;

Ukreślenie koloru jawnie przez atrybut:
[UIStyle(ColorProperty = nameof(PropertyColor))]
public string Property { get; set; }
public VendoColor PropertyColor => VendoColors.Success;

Ukreślenie koloru na stałe przez atrybut:
[UIStyle(Color = VendoColorName.Danger)]
public string Property { get; set; }

Kompletny przykład
VendoUIManager.ShowViewDialog(new MyModel());

class MyModel
{
    [UILabel(ColorProperty = nameof(PropertyLabelColor))]
    public string Property { get; set; }
    public VendoColor PropertyLabelColor => VendoColors.Purple;

    [UIStyle(ColorProperty = nameof(Property2Color))]
    public string Property2 { get; set; } = "Some text, some text.";
    public VendoColor Property2Color => VendoColors.Success;

    [UIStyle(Color = VendoColorName.Danger)]
    public string Property3 { get; set; } = "Some text, some text.";

    public string Property4 { get; set; } = "Some text, some text.";
    public VendoColor Property4Color => VendoColors.Warning;

    public string Property5 { get; set; } = "Some text, some text.";
    public VendoColor Property5Color => VendoColors.Info;
}
